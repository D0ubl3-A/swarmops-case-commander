# Demo Script

Target length: under 5 minutes.

## 0:00 - Hook

AI agents are powerful, but enterprise teams cannot let them run wild. Procurement, finance, HR, and claims teams need speed plus control: case stages, human approvals, and audit-ready evidence.

## 0:30 - Input

Show the business request:

```text
Onboard Northstar Components as a new vendor for a $48,500 rush purchase. Validate risk, prepare approval, and create the procurement handoff.
```

## 1:00 - Case Opens

Show stages:

- Intake
- Vendor Data Review
- Compliance Check
- Purchase Approval
- Fulfillment Handoff
- Final Audit

## 1:30 - Agents Work

Show bounded agents:

- Intake Agent
- Vendor Data Agent
- Compliance Agent
- Approval Summary Agent
- Evidence Agent

Call out that each agent has one job, evidence output, and stop condition.

## 2:20 - Human Approval Gate

Show the risky action blocked:

```text
Approve vendor for purchase setup
```

Approve it manually. Show the case state changing.

## 3:00 - Final Evidence

Show audit report:

- agent outputs
- blocked and approved actions
- risk flags
- evidence IDs
- final case status

## 4:00 - UiPath Fit

Explain mapping:

- UiPath Maestro Case orchestrates stages
- Human tasks handle approvals
- Robots/API steps handle safe automations
- SwarmOps returns plan, risk flags, and evidence JSON

## 4:40 - Close

SwarmOps turns agent chaos into governed enterprise casework: fast enough to matter, controlled enough to trust.
