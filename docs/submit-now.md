## Short Description

SwarmOps Case Commander turns messy enterprise case work into governed agent execution: bounded agents handle safe analysis, risky actions pause for human approval, and every important output becomes audit-ready evidence.

## Long Description

SwarmOps Case Commander demonstrates a vendor onboarding and rush purchase approval workflow. A seeded business request opens a case, creates staged work, delegates safe analysis to five bounded agents, blocks the state-changing vendor setup action for human approval, and generates evidence JSON before and after the approval transition.

The project is designed for the UiPath Maestro Case track. Maestro Case should own stages, human tasks, and governed orchestration, while SwarmOps provides the local agent/evidence layer through a handoff API, OpenAPI contract, and evidence artifact.

Verified implementation includes the local command center prototype, evidence generation, approval-state transition, local handoff API, OpenAPI contract, hosted submission page, presentation deck, narrated demo video artifact, GitHub release, and automated verification scripts.
